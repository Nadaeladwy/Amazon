Dependencies:

Java Development Kit (JDK)
Selenium WebDriver
TestNG (testing framework)
WebDriverManager (for managing WebDriver binaries)
Prerequisites:

Web browser (Chrome, Firefox, etc.)
Internet connection
Installing Dependencies:

Ensure you have Java and Maven installed on your system.
Download the Selenium WebDriver Java bindings and the ChromeDriver executable

Clone this repository to your local machine

Navigate to the project directory
cd amazon-website-tests
Replace "path/to/chromedriver" with the actual path to the ChromeDriver executable in the test script.
Running the Tests:
Open the test plan document to understand the test scenarios and test cases.
Open your preferred web browser.
Navigate to the Amazon website: amazon.com.
Follow the test cases outlined in the test plan document.
